📱 Open on Phone

Your phone needs to open this file from your PC over WiFi so the camera permission works. Follow the steps below.

Step 1 — Start a local web server on your PC
Open Command Prompt (Windows) or Terminal (Mac) and run:
python -m http.server 8080 Keep that window open. If Python isn't installed, download it free from python.org.
Step 2 — Make sure PC and phone are on the same WiFi
Connect both devices to the same WiFi network.
Step 3 — Find your PC's IP address
Windows: Open Command Prompt → type ipconfig → look for IPv4 Address (e.g. 192.168.1.5)
Mac: System Settings → WiFi → Details → IP Address
Step 4 — Scan this QR with your phone
Scan with your Android Camera app or Chrome → opens the tool in phone browser → camera permission will work!

⌨ Keyboard shortcuts & scan guide

Quick reference for fast inventory counting.

Count entry — keyboard

EnterSave count and jump to the next item's count field
TabMove to next field (count → expiry → description → notes)
Shift + TabMove to previous field
0Enter 0 to flag a possible loss (row turns red)

Barcode scanner (USB / Bluetooth)

ScanScanner fires the barcode value + Enter into the Scan input box
EnterTriggers the lookup — jumps to product row and focuses count field
💡 Make sure the dark Scan mode bar is visible at the top of Count Entry. If it says "Scan mode off", click the toggle button to re-enable it.

Navigation buttons

Next uncountedJumps to first item with no count yet and focuses the input
Flag recountMarks an item for a second verification count (yellow row)
Show new items onlyFilters to products added from the latest CSV upload (purple rows)

Row colour guide

Green — item has been counted
Red — count is 0 but system shows stock (possible loss)
Yellow — flagged for recount
Purple — new product added from latest CSV upload
Yellow tint — item expiring within 90 days
